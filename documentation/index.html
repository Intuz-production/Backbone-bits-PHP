    Get Started

    Backone is an amazing tool to pitch your app with the major mobile platforms that is effective for both new and existing apps. Slice and dice your audience from our dashboard analytics with live metrics.

    This documentation will help you to add and configure your apps with Backbone bits tool. You can gain deep insight and easily promote, prompt, respond all of your apps very quickly and with little effort using Backbone bits's iOS and Android SDK. Moreover, you can add FAQ, Videos, Images for the guidance of the app using help module. The SDK integration process is designed to be as easy as possible with a basic setup complete in under 5 minutes.

    Add Apps

    After registering on Backbone bits, it enables you to adding the app while you only introduce company name and suggest automatically your apps. You can also add manually on the screen and simply hitting the save.

    Backbone bits allows two different ways to add your apps inside the system:

    Search with Developer Name / App Name (Only Available for iOS)

    • Select your preferred platform and start searching with Developer Name / App Name
    • Backbone bits will suggest company name from the app store based on your inputs
    • Choose your desired company name and Backbone bits will automatically fetch all the apps listed under that company

    Add app Manually

    • Select your preferred platform and start adding required fields
    • Hit on save button to save app

    App Info

    Select any one app from the apps list. It will show details screen of the app. Moreover, user can refresh the app for getting latest data of app from the apple store by clicking on it. You can update logo, company logo and app images and hit save and publish button to save changes.

    App Settings

    App settings screen will display features list of selected app. As an admin you can enable / disable features of the selected app. Moreover, at below you can allocate work to one of the support agent by clicking on user.

    App Notification

    App notification will allow you to enable / disable notification. If app is iOS then it will ask you to add passphrase & .pem file of Development and production. If app is android then it will ask you to add api key. Moreover, you can set Auto Review message which will be displayed in Respond section.

    Add Help

    Help feature will allow you to add FAQs of current version. Moreover you can upload videos and images. Following will you to add help successfully.

    • Select "+" to add version
    • Select live version and click on "Add FAQ" button to add FAQ
    • Click on "Video" tab and add video of youtube, viemo or manual and hit save & publish
    • Click on "Images" and add images with animation features and click on save and publish
    • To view previous version click on previous version. Moreover, you can view other past versions data from archive list

    Respond List

    Respond list will show the list of support messages of all apps generated by the app user. You will be able to allocate the work to any one of the support agent who can respond to the app users message. You can search messages by using search parameters. Search will help you to get result at very minified level.

    Respond Details

    Respond Detail screen will allow agent / admin to send reply to the app user messages. Moreover, agent / admin can save messages as canned response which can be used later in other response. Agent / admin will also have ability of close the ticket and can ask for review to the app user.

    Setting Details

    Settings section will allow you for basic setup of the apps.

    • You can update your profile details by clicking on profile link.
    • As mentioned in apps section, apps will allow you to add and update the apps.
    • Moreover, you can create agent with any one of the available roles (i.e. Technical, Finance, Support, Marketing, Admin)
    • In access level menu you will be able to view the access given to each roles.
    • Activity log will track down each logs details of the app.
    • General setting feature will allow you to deactivate any of the five features
